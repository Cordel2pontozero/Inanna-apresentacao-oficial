## Publicação no GitHub Pages

1. Envie os arquivos da raiz.
2. Crie `docs/` e coloque os quatro documentos dentro.
3. Abra **Settings → Pages**.
4. Selecione **Deploy from a branch**.
5. Escolha `main` e `/ (root)`.
6. Salve.
